# Begin - Your Personal OS 🚀

> An integrated, beautifully designed personal operating system built with React to streamline student life, productivity, and personal growth.

<img src="https://github.com/user-attachments/assets/fc77e8a2-e523-4083-8264-3ea198ab2363" width="100%" alt="Begin Dashboard Overview" />

[**🌐 View Live Demo**](https://begin-blue.vercel.app/ )

**Begin** is a comprehensive, modular productivity dashboard designed specifically to bring all your essential tools into one cohesive interface. Rather than juggling multiple apps for notes, finances, and study tracking, Begin offers a "Personal OS" experience that unifies your workflow.

## ✨ Key Features

### 📊 Unified Dashboard
A beautifully crafted overview of your day. The dashboard presents real-time data at a glance—from study streaks and recent notes to current financial balances. Features a responsive, mobile-friendly design with dynamic widgets.

### 🍅 Gamified Study Sessions
- **Focus Timer:** Built-in timers optimized for techniques like Pomodoro.
- **Virtual Garden:** Stay motivated by growing flowers for every focus block you complete.

### 📓 Seamless Note-Taking
- **Organized Notebooks:** Categorize your thoughts and class notes into distinct notebooks.
- **Rich Editor:** A distraction-free editor accessible directly from your dashboard.

### 💰 Personal Finance Tracker
- **Budgeting:** Track income and expenses with an intuitive interface.
- **Goal Progress:** Visual progress bars for savings goals and tech upgrades.

### 🎯 Comprehensive Goals Module
- **GPA Tracker:** Log grades and automatically calculate your GPA trajectory.
- **Vision Board:** A visual space to map out long-term goals and stay inspired.

### 🌓 Aesthetic UI/UX
- **Dark/Light Mode:** Seamless theme toggling for comfortable viewing.
- **Modern Design:** Clean typography and smooth transitions for a premium experience.

## 🛠️ Tech Stack
- **Frontend:** React 18, Vite
- **Styling:** Custom CSS & CSS Modules
- **State Management:** Custom React Hooks (`useStore`, `useTheme`)
- **Deployment:** Vercel

## 🚀 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/CarlosJamesDeLeon/begin.git

2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Run the development server:**
   ```bash
   npm run dev
   ```
4. Open `http://localhost:5173` to view it in the browser.
